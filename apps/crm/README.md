# CRM Persia — Cliente

CRM para atendimento via WhatsApp. Roda no EasyPanel em `crm.funilpersia.top`.

## Stack

- Next.js 15 (App Router)
- Supabase (auth + database + realtime)
- Tailwind CSS
- UAZAPI (WhatsApp)

## Deploy

Push em `main` → EasyPanel faz auto-deploy automaticamente.

## Desenvolvimento local

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).
